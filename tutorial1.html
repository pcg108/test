Key Points

  • All tutorials will have a summary of the key points mentioned in the video.
  • C is one of the most powerful and widely used computer programming languages.
  • It is a low-level language, which means that there is less abstraction. You can deal more directly with the processes and memory of the computer.
  • Many other programming languages, such as Java and C++ are developed from C, so learning C makes it easier to learn these and many other languages.
  • Notepad++ is a text-editor we will use to write our C programs. It is similar to text-editors like Notepad, but helps makes our code easier to structure and read.
  • Most all programming languages require a compiler, which takes the code we write and generates the machine-language code that the computer uses.
  • MinGW is a popular C compiler. To install:
    • Download MinGW Installer from website.
    • Run the setup file and click "continue" until you reach the MinGW Installation Manager.
    • In the Installation Manager, select "All Packages" and find the mingw32-gcc option.
    • Right-click and select "mark for installation". Click "Installation" on the top menu, and select "apply changes", and apply.
    • Open your file explorer, navigate to the C:\MinGW\bin folder (which is the default, but if you installed it in another folder, navigate there), and find a file named "gcc.exe".
    • Right-click on the file, select "Properties", and copy the location of the file.
    • Find "PC Info" on your computer. Click on the "Advanced System Settings" option. Select "Environment Variables".
    • Under "System Variables", find the Path variable. Click it, and then click "Edit...".
    • Insert a semicolon at the end of the line, WITHOUT CHANGING ANYTHING ALREADY THERE. Paste the location you copied earlier after the semicolon.
